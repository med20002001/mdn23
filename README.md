#  Nom du projet

Site web moderne construit avec **Astro**, intégrant des composants React, une gestion de formulaires avancée avec **React Hook Form**, une UI élégante grâce à **shadcn/ui**, et une gestion de contenu typée via les **Content Collections** d’Astro.

##  Technologies utilisées

- [Astro](https://astro.build/) – Framework web moderne orienté performance
- [React](https://react.dev/) – Pour les composants interactifs
- [React Hook Form](https://react-hook-form.com/) – Gestion des formulaires et validation
- [shadcn/ui](https://ui.shadcn.com/) – Composants UI accessibles et stylés (basés sur Radix + Tailwind)
- [Tailwind CSS](https://tailwindcss.com/) – Framework CSS utilitaire
- Content Collections (Astro) – Gestion de contenu typée et sécurisée
